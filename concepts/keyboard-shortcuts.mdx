title: "Keyboard Shortcuts"
---

### **Editing and navigation**

- `Enter`: Edit cell or save (if already editing)
- `Shift` **\+**  `Enter`: Open plan picker
- `Cmd (Ctrl)` **\+**  `Enter`: Toggle drill-in in driver tables
- `Cmd (Ctrl) +  (Opt/Alt) `**\+** `Enter`: Collapse this level of drill-in in driver tables
- `Shift` **\+** `Select (or Arrow)`: Select drivers or data cells
- `Option/Alt + Shift + A` - Hide or show the **Actual Formula** column
- `Option/Alt + Shift + F` - Hide or show the **Forecast Formula** column

### **Copy, paste, and undo/redo**

- `Cmd (Ctrl) `**\+ ** `C`: Copy
- `Cmd (Ctrl)` **\+**  `V`: Paste
  - Drivers and data values can both be pasted, but must be done separately
  - To paste into an empty group, select an existing driver first, then use `Cmd (Ctrl)` **\+** `V` to paste below
- `Cmd (Ctrl) `**\+**  `Z`: Undo
- `Cmd (Ctrl) + Shift`\+ `Z`: Redo

### **Driver management**

- `Backspace` (⌫) - Remove driver from block (non-destructive)
- `Shift + Backspace` (⇧ \+ ⌫) - Delete driver entirely (destructive)
- `Cmd (Ctrl) `**\+**  `G`: Group selected drivers
- `Cmd (Ctrl) `**\+**  `D`: Duplicate a driver
- `Cmd (Ctrl) +  Shift`\+  `Enter`: Open selected driver’s detail view
- `Cmd/Ctrl + Option/Alt + =` : Add a driver above the current selection
- `Cmd/Ctrl + Option/Alt + -` : Add a driver below the current selection
- `Cmd/Ctrl + Shift + B` : Add selected drill-in drivers to block
- `Cmd/Ctrl + Enter` : Toggle drill-in state (expand/collapse)
- `Cmd/Ctrl + Option/Alt + Enter` : Collapse parent drill-in from a striped input line

### **Search, context menu, and add**

- `Cmd (Ctrl) `**\+**  `K`: Search
- `Cmd (Ctrl) `**\+**  `/`: Open the context menu directly in a driver grid
- `/`: Add a block on a page
